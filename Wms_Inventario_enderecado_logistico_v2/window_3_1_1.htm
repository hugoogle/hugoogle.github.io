Grid de Manutenção

 

Grid


 

Na grid é ilustrado com campo com o numero do lote, o endereço logístico, o usuario que fechou o lote, quantidade de itens que possui o endereço logístico, quantidade atual, quantidade contada e a diferença.

 

[Top]

Link


 

O campo endereço logístico possui um link onde é aberto um modal para realizar ao ajuste da quantidade do item coletado.

[Top]

Ações


É possivel abrir um lote fechado ou realizar o fechamento do mesmo.
