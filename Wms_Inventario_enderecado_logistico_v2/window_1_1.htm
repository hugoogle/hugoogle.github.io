Data de Competência


A data de competência é a data onde o sistema se baseia para gerar os relatórios de análise de estoques de ganho ou perda.

[Top]

Descrição


É o nome dado ao inventário. Exemplo: Inventário Geral CD-930

[Top]

Tipo


Parcial, é possivel ser gerado a partir de um departamento, seção, grupo, subrupo ou escolhendo determinados produtos.

[Top]

Produto


• Incluir somente produtos com estoque negativo: Nesta opção será gerado o inventário parcial somente de produtos com os estoques negativos, independente do departamento, seção, grupo, subrupo ou produtos selecionados.

• Incluir produtos inativos: Nesta opção será gerado o inventário parcial onde é listado os produtos ativos e inativos para seleção.

• Apenas produtos em linha de loja: Nesta opção será gerado o inventário parcial listando apenas os produtos em linha de loja.
